# Contribuyendo al Proyecto

[English Version](CONTRIBUTING.md)

¡Gracias por considerar contribuir a nuestro proyecto! Damos la bienvenida a las contribuciones de la comunidad y agradecemos su apoyo.

## Cómo Contribuir

1. **Hacer un fork del repositorio**: Haga clic en el botón "Fork" en la esquina superior derecha de la página del repositorio para crear una copia del repositorio en su cuenta de GitHub.

2. **Clonar el repositorio**: Clone el repositorio bifurcado a su máquina local usando el siguiente comando:

   ```
   git clone https://github.com/su-usuario/nombre-repositorio.git
   ```

3. **Crear una nueva rama**: Cree una nueva rama para su contribución usando el siguiente comando:

   ```
   git checkout -b nombre-de-su-rama
   ```

4. **Realizar sus cambios**: Haga los cambios necesarios en el código. Asegúrese de que sus cambios cumplan con los estándares y pautas de codificación del proyecto.

5. **Confirmar sus cambios**: Confirme sus cambios con un mensaje descriptivo usando el siguiente comando:

   ```
   git commit -m "Descripción de sus cambios"
   ```

6. **Subir sus cambios**: Suba sus cambios a su repositorio bifurcado usando el siguiente comando:

   ```
   git push origin nombre-de-su-rama
   ```

7. **Crear un pull request**: Vaya al repositorio original en GitHub y haga clic en el botón "New Pull Request". Proporcione una descripción clara y concisa de sus cambios y envíe el pull request.

## Código de Conducta

Por favor, tenga en cuenta que este proyecto se rige por un [Código de Conducta](CODE_OF_CONDUCT_ES.md). Al participar, se espera que cumpla con este código. Por favor, reporte cualquier comportamiento inaceptable a [email@example.com](mailto:email@example.com).

## Reporte de Problemas

Si encuentra algún problema o tiene sugerencias para mejoras, por favor abra un issue en el repositorio. Proporcione tanto detalle como sea posible para ayudarnos a entender y abordar el problema.
